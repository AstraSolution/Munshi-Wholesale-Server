# Munshi Wholesale Backend

Welcome to Munshi Wholesale Backend! This repository contains the server-side code for our e-commerce website built with the MERN stack (MongoDB, Express.js, React.js, and Node.js). Firebase handles user authentication.

## Technology Stack

- **Backend Framework:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** Firebase

## Project Overview

Munshi Wholesale is a single-vendor e-commerce website. It features user and admin functionalities such as product management, order management, user management, and authentication. The backend provides RESTful APIs to serve data to the frontend, handle authentication, and interact with the database.
